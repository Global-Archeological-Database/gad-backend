# Useful Commands

## Firebase
- `firebase emulators:start` тАФ Start local emulators
- `firebase deploy` тАФ Deploy to production
- `firebase deploy --only functions` тАФ Deploy only Cloud Functions

## GCP
- `gcloud auth login` тАФ Authenticate with GCP
- `gcloud config set project PROJECT_ID` тАФ Set active project

## Node
- `npm run dev` тАФ Start dev server
- `npm run build` тАФ Build for production
- `npm test` тАФ Run tests
