- Created `src/app/submit/page.tsx` — AuthGuard wrapper that opens the submit form on mount
- Modified `src/components/map/MapExplorer.tsx` — Wired FAB button to `setIsSubmitFormOpen(true)`, added `<ArtifactSubmitForm />` component
- Modified `src/app/layout.tsx` — Added `<Toaster>` from sonner for toast notifications
- Installed `sonner` dependency
- Fixed TypeScript issues: `CreateArtifactPayload` type alignment, `@base-ui/react` Select `onValueChange` signature
- Verified: zero TypeScript errors, build compiles successfully

### Notes
- `sonner` was used instead of shadcn `toast` component (which doesn't exist in the registry for this version)
- `@base-ui/react` Select's `onValueChange` has a different signature than typical shadcn Select — takes `(value: string | null, eventDetails)` instead of `(value: string)`
- `CreateArtifactPayload` requires `uploader_name` (not in the Omit list) — passed as `null` since the backend sets it from auth context
- The submit flow is: `create` → get artifact ID → upload files via signed URL → `update` with URLs → invalidate queries → navigate to new artifact detail page
