## Key Decisions
- **SSH blocked on network**: Both port 22 and 443 to GitHub time out. Future git operations should use HTTPS with a Personal Access Token.
- **Artifact Registry over Container Registry**: GCR is deprecated; AR is the standard.
- **Default compute SA for Secret Manager**: Used `65470444307-compute@developer.gserviceaccount.com` (the Cloud Run runtime SA) rather than a dedicated SA, since Cloud Run already runs as this identity.
- **Firestore ordering by `created_at DESC`**: Changed from `__name__` to match deployed composite indexes, enabling filtered queries to work in production.
